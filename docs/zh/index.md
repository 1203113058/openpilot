# openpilot

[openpilot](http://github.com/commaai/openpilot)是一个开源的驾驶辅助系统。openpilot为越来越多的支持的汽车品牌、型号和年款提供自适应巡航控制(ACC)、自动车道居中(ALC)、前方碰撞警告(FCW)和车道偏离警告(LDW)功能。
